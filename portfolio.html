Gravity: Emergent Beauty is a software-art project I started in 2015, in the Processing creative coding environment. Bouncing a ball around the screen is one of the first things I was shown how to code. It's the “hello world” of graphics coding. Fascinated, I started introducing more and more elements and interactions into these sketches, but none were so fruitful as simple newtonian gravity.



Years later this project still captures my attention like nothing else. I re-wrote the software last year in C+SDL, adding new capabilities and improving the interface.



We discuss the question of authorship a lot in the creative code community. I don't discount “artist programs”, but this project is decidedly an art tool. I liken my process to Bowie's method of making lyrics out of magazine clippings, or to laying on the grass, watching clouds go by, trying to “see” things in them, (a dolphin, an elephant...): The software provides the 'stuff' with which I compose a piece.
